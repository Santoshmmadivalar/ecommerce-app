# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/yourusername/ecommerce-App.git
cd ecommerce-App
```

---

## 2️⃣ Move Project

Move project folder to:

```bash
C:/xampp/htdocs/
```

---

## 3️⃣ Start Server

Open XAMPP and start:

* Apache
* MySQL

---

## 4️⃣ Database Setup

* Open phpMyAdmin
* Create database:

```sql
ecommerce
```

* Import:

```bash
database.sql
```

---

## 5️⃣ Configure Database

Open:

```bash
includes/db_connect.php
```

Update credentials:

```php
$host = 'localhost';
$dbname = 'ecommerce';
$username = 'root';
$password = '';
```

---

## 6️⃣ Run Application

```bash
http://localhost/ecommerce-App/
```
